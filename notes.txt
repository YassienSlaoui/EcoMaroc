npm install -g @angular/cli@17 
Generate Commands:

ng g c <component-name>: Generate a new component.
ng g s <service-name>: Generate a new service.
ng g m <module-name>: Generate a new module.
ng g d <directive-name>: Generate a new directive.
ng g p <pipe-name>: Generate a new pipe.
ng g i <interface-name>: Generate a new interface.
ng g cl <class-name>: Generate a new class.
ng g e <enum-name>: Generate a new enum.
ng g g <guard-name>: Generate a new route guard.

Serve & Build Commands:

ng serve: Compile and serve the application, watching for changes.
ng build: Build the application for production.
ng build --prod: Build the application with optimizations for production.
ng test: Run unit tests using Karma.
ng lint: Lint the project files using TSLint.

Miscellaneous Commands:

ng new <project-name>: Create a new Angular project.
ng add <package>: Add an Angular package (e.g., Angular Material).
ng update: Update Angular dependencies and other packages.
ng e2e: Run end-to-end tests using Protractor.
ng doc <keyword>: Opens the official Angular documentation for the keyword.

Configuration & Help:

ng config <key> <value>: Set configuration values in the Angular CLI.
ng help: List available commands and options in the Angular CLI.
